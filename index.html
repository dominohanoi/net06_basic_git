Các bước đẩy code lên github 1. Tạo repository trên web github 2. Clone repository về máy local : git remote add orgin 3. Code trên máy tính: tạo file, sửa file, xóa file,... 4. Chọn những file cần đẩy lên github a. Đẩy hết: git add - kiểm tra trạng thấy git status 5. Push code lên github - commit: git commit -m "Nội dung cần comment code" 6. Đẩy code lên github (với lần đầu: git push -u origin master hoặc main)
